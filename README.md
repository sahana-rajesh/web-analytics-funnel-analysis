# Web Analytics & Conversion Funnel Analysis

## Overview
Analyzed Google's public GA4 e-commerce dataset in BigQuery to identify 
funnel drop-off patterns, traffic source performance, and device revenue trends.

## Key Findings
- 77% of sessions never view a product — largest leak at the discovery stage
- Paid/referral traffic converts at 4.62% — 2.8x higher than direct (1.65%)
- Mobile drives 41% of purchases but has 6% lower AOV than desktop

## Tools
- BigQuery (SQL analysis)
- Looker Studio (dashboard)
- Dataset: `bigquery-public-data.ga4_obfuscated_sample_ecommerce`
